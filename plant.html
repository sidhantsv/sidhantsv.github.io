Automatic Plant Watering System

A simple experiment to test my programming and hardware-software integration skills

A simple personal project I embarked on to test my programming and Arduino board skills, I constructed a very simple automatic plant watering system that would automatically water the plant once it detected a soil moisture level below a certain threshold.

  About the system

This project used a moisture sensor provided with a SEEED Groveboard Kit, a MOSFET and Pump set to automate the process of watering a plant. The hardware utilized parts of Arduino IDE's code and Java to run. The setup sensed the environment and responded with a specific set of actions when conditions were met. The "M" value refers to the Moisture Sensor value.

The primary goal of this system was to water a plant when the soil reaches a specific threshold of moisture. To do this, there were two ways: manual and automatic, and the project uses both methods to effectively water the plant.
